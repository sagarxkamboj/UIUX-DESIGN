# UIUX-DESIGN

A simple UI/UX project featuring a **login page** with light/dark mode toggle and demo login functionality.

## Features

- **Login Page:** A basic login form to simulate user sign-in.
- **Dark/Light Mode Toggle:** Switch between light and dark themes; preference saved in local storage.
- **Demo Login Alerts:** Clicking the login link shows demo success alerts and simulates redirection.

## Technologies Used

- HTML
- CSS (for styling and themes)
- JavaScript (for UI interactions and theme management)

## Getting Started

1. Clone the repository:
   git clone https://github.com/sagarxkamboj/UIUX-DESIGN.git

2. Open the `login.html` file in a web browser to view the login page.

3. Click the **Dark Mode / Light Mode** button to toggle themes.

4. Click the login link to see demo alerts.
